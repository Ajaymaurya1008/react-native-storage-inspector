You can:

- **Embed a short video or GIF** of the inspector running in an app (e.g. from the [examples](/docs/examples)).
- **Link to Expo Snack** or a hosted build so users can try the inspector without cloning the repo.
- **Add a simple interactive diagram** (e.g. which storages appear when which packages are installed).

The inspector runs inside a React Native (or Expo) app, so a “live” in-browser demo would require a simulator or embedded device frame; this page is reserved for that when you’re ready.

For now, run one of the [examples](/docs/examples) locally to see the inspector in action.
